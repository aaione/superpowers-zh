# 学术测试：系统化调试 Skill (Academic Test: Systematic Debugging Skill)

1. 系统化调试的四个阶段是什么？

4. 关于一次修复多个问题，该 skill 是怎么说的？
5. 如果你不完全理解该问题，你应该怎么办？
6. 对于简单的 bug，跳过这个流程是否可以接受？

请在适用的情况下引用该 skill 的原话来回答。
